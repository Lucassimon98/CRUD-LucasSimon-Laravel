## CRUD Laravel - Lucas Simon

Aplicação criada para testar os conhecimentos obtidos em Laravel. Aqui apliquei conceitos como:
Route,
Function,
Controller,
Models,
View,
Um pouco de CSS (Front-End não é o foco nesse projeto),
Conexão com o Banco de Dados,
Autenticação de Login com email e senha,
Layout,
Migrations.

E por fim tudo isso usando sempre o Git para manter o projeto atualizado no repositório.
